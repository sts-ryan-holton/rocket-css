Typography

Use our quick and easy typography helpers in Rocket CSS for changing different aspects of typography.

Basics#

Rocket CSS provides a few useful typography classes for quick, easy application edits.

Text Alignment#

Change the text alignment with these helpers.

  • - .rkt-text-left to set text alignment.
  • - .rkt-text-center to set text alignment.
  • - .rkt-text-right to set text alignment.

Text Transform#

Change the text transform with these helpers.

  • - .rkt-text-lowercase to set text transform.
  • - .rkt-text-uppercase to set text transform.

Font Weight#

Change the font weight with these helpers.

  • - .rkt-font-weight-lighter to set font weight to $font-weight-light
  • - .rkt-font-weight-normal to set font weight to $font-weight-normal
  • - .rkt-font-weight-bold to set font weight to $font-weight-bold

Text Positioning#

Change the text positioning with these helpers.

  • - .rkt-align-top to set vertical alignment.
  • - .rkt-align-middle to set vertical alignment.
  • - .rkt-align-bottom to set vertical alignment.
  • - .rkt-align-baseline to set vertical alignment.
  • - .rkt-align-text-top to set vertical alignment.
  • - .rkt-align-text-bottom to set vertical alignment.
